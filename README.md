# Backup Rival Radar

Small static app for tracking competitors to Backblaze Computer Backup.

## What it does

- Seeds a starter list of researched backup competitors.
- Includes a built-in Backblaze Computer Backup profile as the comparison baseline.
- Opens with seller guidance and value framing instead of dropping directly into editing.
- Lets you type a known company name and autofill pricing, segment, platforms, storage model, features, and notes.
- Lets you add, edit, and delete competitor profiles, including pricing scope, storage scope, retention, restore, security, and line-by-line features.
- Adds seller-facing fields for competitor limitations, `how we win`, and `how we lose` guidance.
- Adds cloud-sync positioning guidance for Dropbox, Google Drive, and iCloud overlap.
- Tracks legal hold, eDiscovery, retention behavior, Linux support, server support, and bare-metal recovery as comparison fields.
- Lets you select one or more competitors for a side-by-side battlecard against Backblaze.
- Exports the selected battlecard as text or opens a print-friendly PDF flow.
- Saves competitor changes in browser `localStorage`.
